# Code Folder

This folder contains scripts and assets used for model inference generation.

## Contents

- `inference.py` - main inference script
- `model.pt` - trained model artifact
- `audio_cues/` - generated audio cues and generation utility
- `region_classes/` - class reference images

## Notes

- Keep script and model versions aligned.
- If you publish results based on this code, cite our IEEE Access paper.
