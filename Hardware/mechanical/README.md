# Mechanical Hardware

Mechanical CAD and manufacturing files for frame prototypes and PCB assemblies.

## Cleanup performed

- Flattened previous `v1_early_prototype/` nesting so files now sit directly in `mechanical/`.

## File types

- `*.SLDPRT`, `*.SLDASM` - SolidWorks part/assembly files
- `*.step`, `*.stp` - neutral CAD exchange files
- `*.STL`, `*.3MF` - printable mesh formats
- `*.DXF` - 2D profile/export format

## Naming notes

Most filenames were already descriptive; they were preserved to avoid breaking downstream references.
